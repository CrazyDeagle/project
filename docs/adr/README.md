# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the
SilexCode project. The format and rationale are described in
[ADR-0001](0001-record-architecture-decisions.md).

## Index

| #    | Title                                                              | Status   |
| ---- | ------------------------------------------------------------------ | -------- |
| 0001 | [Record architecture decisions](0001-record-architecture-decisions.md) | Accepted |

## Writing a New ADR

1. Copy ADR-0001 to `NNNN-short-kebab-title.md` with the next available
   number.
2. Fill in the four sections: Context, Decision, Consequences. Keep it
   short — one page is usually enough.
3. Add a row to the table above.
4. Reference the ADR from the PR description that implements it.
